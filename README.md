📊 Análisis de Clientes aprobados: Exploración y perfil de clientes
📖 Descripción
Este proyecto realiza un análisis exploratorio de las variables que muestran patrones de estudiantes. El objetivo es identificar perfiles y patrones para entender que variables pueden afectar al rendimiento academico.
🗂️ Estructura del Proyecto
	-CSV
	-Excel con la transformación , análisis y dashboard


🛠️ Instalación y Requisitos
Este proyecto usa Excell
📊 Resultados y Conclusiones
•	El rendimiento académico está dominado por hábitos previos y actuales del estudiante: Previous_Scores + Horas estudiadas + Asistencia.
•	Los recursos y el entorno escolar (Calidad del profesor y el acceso a los recursos) tienen un impacto significativo y consistente.
•	La motivación es un factor psicológico clave que modula el efecto de las demás variables.
•	El contexto familiar influye, pero menos que los hábitos del estudiante.
•	Variables como género, distancia o tipo de escuela tienen un impacto muy pequeño en tu dataset
